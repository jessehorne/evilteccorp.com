Useful Endpoints

Get Token

Some endpoints require an Authorization header of `YOUR TOKEN`. Tokens are only valid for 1 hour.
POST https://evilteccorp.com/api/token
{
    "email": "YOUR_EMAIL_HERE",
    "password": "YOUR_PASSWORD HERE"
}
